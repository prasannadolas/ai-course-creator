**Quiz: Arrays and Linked Lists**

1.  **Which data structure provides **O(1)** access time to elements?**
    a)  Linked List
    b)  Array
    c)  Both
    d)  Neither

2.  **What is the time complexity of inserting an element into the *middle* of a Python list (array)?**
    a)  O(1)
    b)  O(log n)
    c)  O(n)
    d)  O(n^2)

3.  **True or False: Linked lists store elements in contiguous memory locations.**
    a) True
    b) False

---

**Answer Key:**

1.  b)
2.  c)
3.  b)
