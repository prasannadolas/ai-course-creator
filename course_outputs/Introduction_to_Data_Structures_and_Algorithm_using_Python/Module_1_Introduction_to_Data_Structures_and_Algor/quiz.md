**Quiz: Introduction to Data Structures and Algorithms**

1.  **Which of the following best describes a data structure?**
    a)  A step-by-step procedure to solve a problem.
    b)  A specific way of organizing and storing data in a computer for efficient use.
    c)  A set of mathematical formulas.
    d)  The hardware components of a computer.

2.  **What is the primary role of an algorithm?**
    a)  To store data.
    b)  To define the structure of a database.
    c)  To perform a specific task or solve a problem through a series of steps.
    d)  To manage the computer's operating system.

3.  **True or False: The choice of data structure always impacts the efficiency of algorithms that operate on it.**
    a) True
    b) False

---

**Answer Key:**

1.  b)
2.  c)
3.  a)
