**Quiz: Trees and Graphs**

1.  **Which of the following is NOT a characteristic of a tree data structure?**
    a)  Nodes
    b)  Edges
    c)  Cycles
    d)  Root

2.  **Which graph representation is more space-efficient for sparse graphs (graphs with few edges)?**
    a)  Adjacency Matrix
    b)  Adjacency List
    c)  Both are equally efficient
    d)  Neither are efficient

3.  **True or False: Breadth-First Search (BFS) explores a graph by going as deep as possible along each branch before backtracking.**
    a) True
    b) False

---

**Answer Key:**

1.  c)
2.  b)
3.  b)
