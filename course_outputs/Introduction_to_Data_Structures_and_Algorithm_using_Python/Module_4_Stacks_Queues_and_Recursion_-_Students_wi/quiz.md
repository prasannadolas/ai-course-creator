**Quiz: Stacks, Queues, and Recursion**

1.  **Which of the following principles does a stack adhere to?**
    a)  FIFO (First-In, First-Out)
    b)  LIFO (Last-In, First-Out)
    c)  Both FIFO and LIFO
    d)  Neither FIFO nor LIFO

2.  **What is the purpose of the base case in a recursive function?**
    a)  To perform the main calculation of the function.
    b)  To call the function recursively with modified inputs.
    c)  To stop the recursion and provide a final result.
    d)  To initialize the variables used in the function.

3.  **True or False: Using a Python list for queue implementation, where the head is the beginning of the list and the tail is the end, provides the most efficient enqueue and dequeue operations.**
    a) True
    b) False

---

**Answer Key:**

1.  b)
2.  c)
3.  b)
