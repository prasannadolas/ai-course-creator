1.  What is the purpose of a **communication protocol** in agent systems?
    a) To define the hardware specifications of the agents.
    b) To establish a common set of rules for message exchange.
    c) To manage the internal memory of each agent.
    d) To determine the agents' goals.

2.  Which of the following is NOT a key component of a message used in **message passing**?
    a) Sender
    b) Receiver
    c) Content
    d) Internal State

3.  In a **Multi-Agent System (MAS)**, what is the primary goal of **coordination**?
    a) To ensure each agent acts independently.
    b) To prevent any form of communication between agents.
    c) To enable agents to work together effectively towards a shared goal.
    d) To maximize the individual performance of each agent, regardless of others.

---
**Answer Key**

1.  b)
2.  d)
3.  c)
